Sobre o Projeto

Este projeto foi desenvolvido na disciplina de Processamento de Imagens da UFMA, combinando técnicas avançadas de visão computacional com aprendizado profundo para identificação de espécies vegetais.

O sistema utiliza redes neurais convolucionais e técnicas de processamento de imagens para fornecer identificações precisas e confiáveis.
